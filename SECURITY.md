# Security Policy

## Supported Versions

As this is currently a private development project, we support the latest version of the code.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please report it via the [GitHub Issues](https://github.com/Dex-code61/edge.runner/issues) tab with a "Security" label or prefix.

Please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Any relevant logs or output.

We will try to review and address the issue as soon as possible.
